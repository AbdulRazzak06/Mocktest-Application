📝 Online Mock Test Application
A desktop-based mock test application built using Python that allows users to take timed quizzes on different subjects like JEE, NEET, and Civil Services.

🚀 Features
Select subject (JEE, NEET, Civil Services)
Choose number of questions
Fetches questions from online API
Multiple-choice questions (MCQs)
Timer-based test system
Navigation (Next / Previous questions)
Score calculation with feedback
Displays correct answers after submission

🛠️ Technologies Used
Python
Tkinter
Requests (API handling)
HTML parsing (for decoding questions)

⚙️ Installation
pip install requests

▶️ How to Run
Download the project files
Make sure image.png is in the same folder
Run the Python file:
python filename.py

💻 How It Works
User selects a subject and number of questions
Questions are fetched from an online API
Each question is displayed with options
Timer starts automatically
User navigates and selects answers
Final score and correct answers are shown

📌 Key Functionalities
Real-time countdown timer
Auto-submit when time ends
Dynamic question loading
Scrollable result window
Performance suggestion based on score
